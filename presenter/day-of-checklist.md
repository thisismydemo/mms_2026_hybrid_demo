# Day-Of Checklist — The Hybrid Update Blues

Complete 30 minutes before session start.

## Environment Verification

- [ ] Azure portal loads and all blades are accessible
- [ ] Arc-enabled servers show "Connected" status
- [ ] Azure Local cluster is healthy and accessible (if demoing live)
- [ ] Maintenance configuration exists with dynamic scopes
- [ ] Azure Update Manager shows machines with compliance data
- [ ] Resource Graph queries return expected results
- [ ] Hotpatch-enrolled machine visible with update history
- [ ] Azure Policy compliance data is populated
- [ ] Cost Management blade accessible

## Browser Setup

- [ ] Clear all browser tabs
- [ ] Open bookmarked tabs for each demo starting point:
  - Azure Update Manager (Overview)
  - Azure Update Manager (Machines)
  - Maintenance Configuration
  - Arc-enabled server
  - Azure Local cluster → Updates
  - Resource Graph Explorer (with saved queries)
  - Azure Policy → Compliance
  - Cost Management → Cost Analysis
- [ ] Set browser zoom to 125–150%
- [ ] Disable browser notifications
- [ ] Disable OS notifications (Focus Assist / Do Not Disturb)

## Demo Machine

- [ ] Test projector / screen share with portal open
- [ ] Verify font size readable from back of room
- [ ] Close unnecessary applications
- [ ] Mute phone and messaging apps

## Fallback Materials

- [ ] Backup PowerPoint with screenshots accessible offline
- [ ] Fallback recordings playable
- [ ] `assets/screenshots/` folder current with latest captures
