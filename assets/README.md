# Assets

Store demo visual assets here.

## Directories

| Folder | Purpose |
|---|---|
| `diagrams/` | Architecture diagrams, flow charts, Visio/draw.io exports |
| `screenshots/` | Portal screenshots for fallback slides |
| `recordings/` | Short screen recordings of demo steps as backup |

## Naming Convention

Use descriptive, kebab-case names with a number prefix:
- `01-azure-update-manager-overview.png`
- `02-maintenance-config-settings.png`
- `03-arc-server-connected.png`

## Tips

- Capture screenshots at **1920x1080** for consistency
- Use **PNG** for screenshots, **MP4** for recordings
- Update screenshots if portal UI changes before the session
